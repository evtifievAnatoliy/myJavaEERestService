Пример RESTful сервиса

Краткое описание сервиса.
Во время своей работы сервис обеспечивает запись сообщений пользователей в строковом формате. Каждое сообщение имеет вид "имя_пользователя:сообщение".

Имя пользователя, от имени которого вводятся сообщения, определяется командой "Регистрация". После успешной регистрации на следующей форме пользователь может определять параметры и выполять запросы "Добавить сообщение", "Изменить сообщение", "Удалить сообщение" и "Получить все сообщения".

Для выхода из приложения или смены имени пользователя следует выполнить команду "Выйти".
